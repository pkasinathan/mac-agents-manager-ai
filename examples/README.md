# Examples

This directory contains sample scripts and plist configurations that demonstrate
how to use macOS LaunchAgents. They are **not** part of the core
`mac-agents-manager` package and are provided purely as references.

| File | Description |
|------|-------------|
| `stay_focus.sh` | A productivity script that opens/closes apps on a schedule |
| `user.automation.stay_focus.plist` | Sample plist that runs `stay_focus.sh` at specific times |
| `manage_services.sh` | A standalone Bash service manager for LaunchAgents |
